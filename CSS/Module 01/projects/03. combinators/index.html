Welcome to the Selectors Module

In this module we will discuss the follwing:

  • Simple Selectors
  • Attribute Selectors
  • Combinators

Simple Selectors

Type Selector (name selector)

        h3 - selects all elements of type h3

Class Selectors

        .class-name - selects all elements that contain the class 'class-name'

Id Selector

        #id-name - selects "all" (single) element with the id '#id-name'

Universal Selector

        * - selects all the elements

Descendant Combinator

        parent child - Select all "child" elements that are descendant of "parent"

Direct Child Combinator

        parent > child - Select all "child" elements that are Direct Child of "parent"

The "And" Combinator

        .parent.child - select all elements with class "parent" and also class "child"

General Sibling Selector

        a ~ b - select all b elements that are preceded by a

Direct Sibling Selector

        a + b - select the first b that is placed immediately after a
